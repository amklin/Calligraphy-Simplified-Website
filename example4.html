Canvas Instructions

To use the canvas, click on settings in the upper right corner of the screen.

Each of the buttons that pops up toggles a different setting.

clear: Clears the canvas. You can also use the escape key to clear the canvas.

edit: Turns the canvas on and off without erasing what is currently on the canvas.

line_weight: Adjusts the thickness of the line.

color_lens: Change the color of the brush

The canvas default is off. To turn on the canvas for the first time use either the clear or edit button.

Warning: Don't change the window size while writing. If you change the window size, use the reset button to ensure that everything functions correctly.
